## 用法

### 剪貼簿
讀取從剪貼簿複製的圖片

### 調整座標
左鍵設定開始的座標 右鍵設定邊界的座標防止游標點到其他東西

### 快速開始
按下開始後會3秒會從游標所在的位置開始, 不受邊界限制
